# Hands-On Labs / Exercises

This section lists practical exercises for each module. Students are encouraged to complete these labs to reinforce learning.

## Module 1: ROS 2
- Create a basic ROS 2 node controlling a robot joint  
- Publish and subscribe to topics  
- Implement simple services and actions  

## Module 2: Gazebo & Unity
- Simulate a bipedal robot in Gazebo  
- Add sensors like LiDAR and Depth Cameras  
- Visualize the robot in Unity and test basic movements  

## Module 3: NVIDIA Isaac
- Set up Isaac Sim and train perception pipelines  
- Use Isaac ROS for SLAM and navigation  
- Implement reinforcement learning for robot control  

## Module 4: Vision-Language-Action
- Integrate OpenAI Whisper for voice commands  
- Translate natural language instructions into ROS 2 actions  
- Simulate object recognition and manipulation tasks
